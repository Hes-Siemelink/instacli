# Command: Wait

`Wait` just waits a while

| Content type | Supported |
|--------------|-----------|
| Value        | yes       |
| List         | no        |
| Object       | no        |

[Wait.schema.yaml](schema/Wait.schema.yaml)

## Basic usage

Use **Wait** to pause for the amount of seconds specified.

```yaml instacli
Code example: Wait a tenth of a second

Wait: 0.1
```
